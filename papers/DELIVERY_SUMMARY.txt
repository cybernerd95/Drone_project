TROUBLESHOOTING
================================================================================

MOST COMMON ISSUES:

Q: "Undefined function or variable 'multiDroneMotion'"
A: Ensure multiDroneMotion.m is in same folder → check working directory

Q: "Out of Memory" error
A: Reduce totalDuration to 2 sec, or array size from [8 8] to [4 4]

Q: Phased Array Toolbox not found
A: Type >> ver to check installed toolboxes
   Add it via MATLAB Add-Ons if available

Q: Simulation takes forever (> 1 hour)
A: Use Quick Test config (totalDuration=1, Ndrones=2)

Q: All plots are blank
A: Check PRF setting (should be 2e3-10e3) or enable debug output

FULL TROUBLESHOOTING: See QUICKSTART.md "Troubleshooting Guide"
